What is the Disturbed Blog


The Disturbed Blog is a web log hosting site designed to allow users to create their own and view other user's blogs

How does this site work?

To use this site, you must first create an account. Once you do, you will be able to access this web service.
After logging in, you will be brought to the home page. From there you have two buttons to either create or edit your posts. Below those options you can view all the blogs created from other users sorted the date added.
  • Creating New Blogs:
    • Each new blog requires two elements:
    • Title: this is the title of your post. This is what viewers first see on the home and view page of the site so make it pop!
    • Date: blog posts on the home page and on your personal page are sorted by date added.
  • Editing Blogs:
    • You are able to edit the title and contents of every post you make.
